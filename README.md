# 🧑‍💻 User Management System

A full-stack **User Management System** built with **Next.js** and **Flask**, featuring secure authentication, role-based access control, and a modern admin dashboard.

## 🚀 Features

* 🔐 Secure Login & Email OTP Verification
* 🔑 JWT Authentication & Protected Routes
* 👥 Role-Based Access Control (RBAC)
* 🛡️ Permission-Based Authorization
* 📊 Modern Responsive  Dashboard
* ⚡ REST API Integration
* 🗄️ Database-Driven User Management

## 🛠️ Tech Stack

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* Flask (Python)
* REST APIs

### Database

* SQLite

### Authentication

* JWT
* Email OTP Verification

## 📌 Project Overview

This project demonstrates a production-style **User Management System** with secure authentication and a flexible permission system. Users authenticate using email OTP verification before accessing the application. The system implements **Role-Based Access Control (RBAC)** and **Permission-Based Authorization**, enabling secure management of users, roles, and application access.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Hammy5959/user-management-system-pro.git
cd user-management-system-pro
```

### 2. Backend Setup

```bash
cd backend
python -m venv .venv
```

**Activate Virtual Environment**

**IF Windows**

```bash
.venv\Scripts\activate
```

**Install Dependencies**

```bash
pip install -r requirements.txt
```

**Run Backend**

```bash
python main.py
```

### 3. Frontend Setup

```bash
cd ../frontend
npm install
npm run dev
```

### 4. Open the Application

Visit:

```text
Frontend: http://localhost:3000/login
```
